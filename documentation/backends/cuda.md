# CUDA 后端

CUDA 后端是 Chomsky 针对高性能并行计算的核心组件。

## 优化策略

- **线程束 (Warp) 友好化**：以分支散布、占用率、同步点为主要成本信号。
- **内存层次具体化**：在具体化层决定寄存器/共享内存/全局内存的布局与搬运策略。
- **内核融合 (Kernel Fusion)**：在约束允许时，通过等价饱和产生融合候选并由成本模型选择。

## 目标硬件

支持所有具备 Compute Capability 6.0 及以上的 NVIDIA GPU。

## 当前实现状态（诚实基线）

当前实现以演示级 PTX 生成为主，覆盖的 iKun 子集有限；完整的规约树、块级并行与内存具体化仍在演进中。
