# 核心输入领域 (Intake Domains)

Chomsky 作为一个通用的计算寻优引擎，通过将不同领域的语义映射到 **Universal IR (IKun)**，实现跨领域的寻优能力。

目前主要支持以下三个核心领域的输入：

## 1. 编程语言领域 (Programming Languages)
**代表项目**：[nyar-vm](../../nyar-vm)
- **侧重点**：控制流图 (CFG) 的拓扑重组、对象模型的去虚化、跨函数边界的内联优化。
- **寻优目标**：最小化执行指令数、减少堆分配、提高 CPU 分支预测准确率。

## 2. 深度学习领域 (Deep Learning)
**代表项目**：[dxo-rs](../../dxo-rs)
- **侧重点**：算子融合 (Operator Fusion)、内存布局变换 (Layout Transformation)、硬件特定指令映射（如 Tensor Core）。
- **寻优目标**：最大化吞吐量、最小化显存带宽占用、降低计算延迟。

## 3. 数据库领域 (Databases)
**代表项目**：[yydb](../../yydb), [yyds](../../yyds), [rbq-engine](../../rbq-engine)
- **侧重点**：逻辑算子下沉、连接顺序重排 (Join Reordering)、向量化搜索寻优、存储布局适配。
- **寻优目标**：最小化磁盘 I/O、减少扫描行数、优化内存利用率。

## 4. 分布式调度领域 (Distributed Systems)
- **侧重点**：任务拓扑优化、计算与数据对偶性寻优、服务折叠、协议自动提取。
- **寻优目标**：最小化网络延迟、降低出口流量成本、最大化系统可用性。

## 5. 硬件综合领域 (Hardware/EDA)
- **侧重点**：重定步 (Retiming)、流水线自动切分、算子强度削减、功耗优化。
- **寻优目标**：最大化时钟频率 (Fmax)、最小化芯片面积 (Area)、降低功耗 (TDP)。
