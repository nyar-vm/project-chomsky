# 编程语言领域 (Programming Languages)

在编程语言领域，Chomsky 充当高级中端优化器的角色，主要对接 [nyar-vm](../../nyar-vm) 等运行时。

## 核心用法
1. **意图提取**：通过 `oak-rust` 或其他前端，将 AST 转化为具有严格所有权和效应语义的 `ICon` 序列。
2. **等价搜索**：
   - **闭包消除**：搜索闭包是否可以重写为静态函数或内联结构体。
   - **循环展开与向量化**：在意图层面识别数据并行性，并尝试重写为 `Map` 原子。
3. **成本评估**：基于目标架构（如 RISC-V 或 Gaia VM）的周期模型进行提取。

## 侧重点
- **上下文敏感性**：重点解决跨函数、跨模块的寻优，消除传统分段编译带来的阻碍。
- **所有权跟踪**：利用 Chomsky 的所有权格系统，自动寻找将堆分配重写为栈分配的机会。
- **动态性消除**：通过等价重写，将接口调用（VTable）转化为直接调用。
